## ğŸ›  Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Ak-AhmedFathi/CRM-Dashboard-UI.git
cd CRM-Dashboard-UI
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

### 4. Build for production

```bash
npm run build
npm run start
# or
yarn build
yarn start
```
